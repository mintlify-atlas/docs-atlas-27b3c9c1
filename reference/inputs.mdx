## KMS

<ParamField name="create_kms_key" type="bool" default="true">
  Controls if a KMS key for cluster encryption should be created.
</ParamField>

<ParamField name="kms_key_description" type="string" default="null">
  The description of the key as viewed in AWS console.
</ParamField>

<ParamField name="kms_key_deletion_window_in_days" type="number" default="null">
  The waiting period, specified in number of days. After the waiting period ends, AWS KMS deletes the KMS key. If you specify a value, it must be between `7` and `30`, inclusive. If you do not specify a value, it defaults to `30`.
</ParamField>

<ParamField name="enable_kms_key_rotation" type="bool" default="true">
  Specifies whether key rotation is enabled.
</ParamField>

<ParamField name="kms_key_rotation_period_in_days" type="number" default="null">
  Custom period of time between each key rotation date. If you specify a value, it must be between `90` and `2560`, inclusive. If you do not specify a value, it defaults to `365`.
</ParamField>

<ParamField name="kms_key_enable_default_policy" type="bool" default="true">
  Specifies whether to enable the default key policy.
</ParamField>

<ParamField name="kms_key_owners" type="list(string)" default="[]">
  A list of IAM ARNs for those who will have full key permissions (`kms:*`).
</ParamField>

<ParamField name="kms_key_administrators" type="list(string)" default="[]">
  A list of IAM ARNs for key administrators. If no value is provided, the current caller identity is used to ensure at least one key admin is available.
</ParamField>

<ParamField name="kms_key_users" type="list(string)" default="[]">
  A list of IAM ARNs for key users.
</ParamField>

<ParamField name="kms_key_service_users" type="list(string)" default="[]">
  A list of IAM ARNs for key service users.
</ParamField>

<ParamField name="kms_key_source_policy_documents" type="list(string)" default="[]">
  List of IAM policy documents that are merged together into the exported document. Statements must have unique `sid`s.
</ParamField>

<ParamField name="kms_key_override_policy_documents" type="list(string)" default="[]">
  List of IAM policy documents that are merged together into the exported document. In merging, statements with non-blank `sid`s will override statements with the same `sid`.
</ParamField>

<ParamField name="kms_key_aliases" type="list(string)" default="[]">
  A list of aliases to create. Note - due to the use of `toset()`, values must be static strings and not computed values.
</ParamField>
