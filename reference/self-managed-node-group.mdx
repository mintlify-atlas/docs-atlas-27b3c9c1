---
title: Self-Managed Node Group
description: Input variables and outputs for the self-managed-node-group sub-module.
---

The `self-managed-node-group` sub-module manages EC2 Auto Scaling Groups for use as EKS worker nodes. It is invoked for each entry in the `self_managed_node_groups` map on the root module.

## Inputs

### General

<ParamField name="create" type="bool" default="true">
  Determines whether to create self managed node group or not.
</ParamField>

<ParamField name="tags" type="map(string)" default="{}">
  A map of tags to add to all resources.
</ParamField>

<ParamField name="region" type="string" default="null">
  Region where the resource(s) will be managed. Defaults to the Region set in the provider configuration.
</ParamField>
