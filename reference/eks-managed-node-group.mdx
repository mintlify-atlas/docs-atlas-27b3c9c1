---
title: EKS Managed Node Group
description: Input variables and outputs for the eks-managed-node-group sub-module.
---

The `eks-managed-node-group` sub-module manages [EKS managed node groups](https://docs.aws.amazon.com/eks/latest/userguide/managed-node-groups.html). It is invoked for each entry in the `eks_managed_node_groups` map on the root module.
