title: Fargate Profile
description: Input variables and outputs for the fargate-profile sub-module.

The `fargate-profile` sub-module manages [EKS Fargate Profiles](https://docs.aws.amazon.com/eks/latest/userguide/fargate.html). It is invoked for each entry in the `fargate_profiles` map on the root module.

## Inputs

### General

  Determines whether to create Fargate profile or not.
</ParamField>

<ParamField name="tags" type="map(string)" default="{}">
  A map of tags to add to all resources.
</ParamField>

<ParamField name="region" type="string" default="null">
  Region where the resource(s) will be managed. Defaults to the Region set in the provider configuration.
</ParamField>

<ParamField name="iam_role_permissions_boundary" type="string" default="null">

## Outputs

### IAM Role

<ResponseField name="iam_role_name" type="string">
  The name of the IAM role.
</ResponseField>

<ResponseField name="iam_role_arn" type="string">
  The Amazon Resource Name (ARN) specifying the IAM role.
</ResponseField>

<ResponseField name="iam_role_unique_id" type="string">
  Stable and unique string identifying the IAM role.
</ResponseField>

### Fargate Profile

<ResponseField name="fargate_profile_arn" type="string">
  Amazon Resource Name (ARN) of the EKS Fargate Profile.
</ResponseField>

</ResponseField>

</ResponseField>
